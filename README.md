# Personal Skills

A collection of personal agent skills for use with [Claude Code](https://claude.ai/claude-code).

## Installation

Add this plugin to your Claude Code configuration:

```bash
claude plugins add dcramer/skills
```

## Available Skills

| Skill | Description |
|-------|-------------|
| `testing-guidelines` | Guide for writing tests. Emphasizes integration tests, real-world fixtures, and regression coverage. |

## Usage

Invoke skills with the slash command:

```
/testing-guidelines
```
